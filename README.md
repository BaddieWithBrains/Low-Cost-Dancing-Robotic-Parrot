# 🦜 Bio-Inspired Dancing Robotic Parrot

## 📌 Project Overview
This project presents the design and implementation of a bio-inspired robotic parrot capable of reacting to audio input and performing rhythmic motion. The system combines mechanical design, embedded control, and signal processing to mimic bird-like movement.

---

## ⚙️ System Description

The robotic parrot consists of:

- 3D-printed structures
- Servo-actuated wings and body
- An ESP32 microcontroller for control
- A MAX4466 microphone module for audio input

---

## 🎵 Functionality

The system detects audio amplitude in real time and performs:

- Body bobbing motion
- Wing flapping (up/down)
- Alternating "seesaw" wing movement
- Tempo-adaptive dancing behaviour

---

## 🔌 Hardware Components

- ESP32 microcontroller
- MAX4466 microphone amplifier
- Servo motors (body + wings)
- 2x 18650 battery system with BMS
- Buck converter for voltage regulation

---

## 🧠 Software Features

- Real-time audio sampling
- Beat detection algorithm
- Tempo adaptation
- Multi-phase motion control

---

## 🛠️ Tools Used

- Fusion 360 (CAD design)
- Arduino IDE (ESP32 programming)
- 3D Printing prusa printing machine
